**فرانت‌اند:** Next.js
**بک‌اند:** Node.js
**دیتابیس:** PostgreSQL

**ویژگی‌های سایت:**

- کروسل‌های روان در صفحه اصلی
- دیدن محصولات
- اضافه کردن به سبد خرید در لوکال استوریج
- سبد خرید
- صفحه محصول
- کامنت‌های محصول
- ایجاد کامنت برای محصول با امتیاز دهی
- اضافه کردن محصول به علاقه‌مندی‌ها در لوکال استوریج
- ثبت نام و ورود
- اعتبار سنجی‌های مختلف در ثبت نام و لاگین با UI‌های مناسب هم در بک‌اند هم فرانت
- اتصال به چت‌بات‌های هوش مصنوعی (فعلاً چت‌بات رایگان)
- داشبورد کاربر
- ایجاد آدرس‌های مختلف
- ویرایش آدرس
- تغییر آدرس پیش‌فرض
- حذف آدرس
- تغییر رمز عبور
- صفحه محصول
- صفحه محصولات با فیلترهای مختلف
  - فیلترها: قیمت، امتیاز، دسته‌بندی
- ترتیب نمایش: پیش‌فرض (جدیدترین)، قیمت کم به زیاد، قیمت زیاد به کم، محبوب‌ترین (بر اساس امتیاز)
- سرچ محصول از طریق سرچ بار و نمایش محصولات به صورت لایو
- پیجینیشن محصولات
- ویرایش اطلاعات کاربری
- تایید ایمیل
- ملاحظه و رعایت نقاط قوت و ویژگی‌های Next.js برای سئو صفحات
- شناخت اعتبار سنجی کاربر با JS Cookie

User dashboard

Edit addresses

Change default address

Product page

Product listing page with various filters

Filters: price, rating, category (most filtering logic is handled on the backend; products were pre-downloaded from APIs, so they lack additional filter-specific attributes)

Sort options: default (newest), price low to high, price high to low, most popular (based on rating)

Live product search via search bar

Pagination for products

Edit user profile information

Email verification

Consideration and implementation of Next.js strengths and best practices for SEO
